Chronos MCP

Cron-based scheduler for MCP tool calls

What is this?

Chronos MCP is a cron-based scheduler that triggers MCP (Model Context Protocol) tool calls at configured times.

It connects to any MCP-compatible server via stdio or HTTP (Streamable) transports and invokes tools on a schedule defined by cron expressions.

Key features:

  • Schedule any MCP tool call with cron expressions
  • Supports stdio and HTTP (Streamable) transports
  • Template variables like {{now}}, {{date}}, {{time}} resolved at trigger time
  • Execution logging with history
  • This server also exposes its own MCP interface, so LLMs can manage cron rules via MCP
